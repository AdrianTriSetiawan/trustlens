# Trust Lens

Lightweight trust report for installed VS Code extensions.

## Features
- Scan installed extensions and show a risk level
- Risk heuristics: startup activation, workspace access, Node.js host
- Quick actions: disable or uninstall
- Export report to JSON or CSV

## Notes
Risk assessment is heuristic. It does **not** guarantee safety.

## Commands
- `Trust Lens: Refresh`
- `Trust Lens: Disable Extension`
- `Trust Lens: Uninstall Extension`
- `Trust Lens: Open Extension Details`
- `Trust Lens: Export Report (JSON)`
- `Trust Lens: Export Report (CSV)`
